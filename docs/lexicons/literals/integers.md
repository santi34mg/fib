# Integer Literal

An integer literal represents an integer constant.
Underscores may be used to visually separate the literal.

**Examples**:

```
123
015
100_000
```

## Differnt bases

Different bases may be specified by adding a prefix (`0b`, `0o`, `0x`).

**Examples**:

```
0xAB12
0b0110
0o7214
```
